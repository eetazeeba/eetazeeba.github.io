Musifer - Frontend project

Overview
- This repo currently tracks the straightforward static site (`index.html`, `CSS/styles.scss`, `CSS/styles.css`).
- Active architecture direction is Jamstack with a file-based CMS model.

CMS direction
- Content targets: blog entries, artist/collaborator profiles, and lesson content.
- Shared metadata priorities: publishing/edit dates, author, copyright, media types, and tags.
- Decap CMS is the leading free candidate and has a clear integration path with Eleventy if the 11ty track is reactivated.
