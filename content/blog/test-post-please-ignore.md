# Welp, I have gone and done it

Instead of keeping things straightforward and easy, here I go making a whole back end to manage what should have been a reasonably simple project. I'm going to keep it up because it seems to be going well so far, but this combo of 11ty and Decap CMS is giving me a good brain workout. Eventually things may need to be reduced to some WYSIWYG editors and a move to having the site hosted by an established platform instead of this "roll my own" model.
